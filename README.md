# Auric Krystals - Premium Crystal & Astrology E-Commerce Platform

A beautiful, responsive website for browsing premium crystals, gemstones, healing bracelets, and astrology services with an interactive crystal quiz.

## 🌟 Features

- **Interactive Crystal Quiz** - 7-question quiz that recommends crystals based on user preferences
- **Product Gallery** - Browse crystals, bracelets, and gemstones with filters
- **Astrology Services** - Kundali readings, horoscope analysis, compatibility checks, ritual guidance
- **Contact Page** - Get in touch with social media integration
- **Responsive Design** - Mobile-first design that works on all devices
- **Premium Design System** - Lavender & gold color palette with glassmorphic surfaces

## 📱 Pages

- **Home** (`index.html`) - Editorial homepage with key attractions and featured products
- **Shop** (`shop.html`) - Product gallery 
- **Product Detail** (`product.html`) - Premium product information page template
- **Services** (`services.html`) - Astrology and wellness services
- **Contact** (`contact.html`) - Contact form and social media links

## 🛠️ Technology Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling via CDN
- **JavaScript (Vanilla)** - Quiz logic with localStorage persistence
- **Google Material Symbols** - Icon library
- **Google Fonts** - Noto Serif & Manrope typography

## 🚀 Deployment on GitHub Pages

### Step 1: Create Repository on GitHub
1. Go to [github.com/new](https://github.com/new)
2. Name it `auric-krystals`
3. Set to **Public**
4. **Do NOT** initialize with README/gitignore
5. Click **Create repository**

### Step 2: Push to GitHub
```bash
git config user.email "your@email.com"
git config user.name "Your Name"
git add .
git commit -m "Initial commit: Auric Krystals website"
git remote add origin https://github.com/YOUR-USERNAME/auric-krystals.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / `(root)`
4. Click **Save**

Your site will be live at: `https://YOUR-USERNAME.github.io/auric-krystals`
